# Bilhete Magico Pro

Aplicacao web em React com Next.js para gerar jogos da Mega-Sena com filtros simples de tendencia, historico local, favoritos, copia dos jogos e exportacao em SVG.

## Tecnologias

- React
- Next.js
- CSS

## Como Executar

1. Instale as dependencias:

```bash
npm install
```

2. Inicie o servidor de desenvolvimento:

```bash
npm run dev
```

3. Abra no navegador:

```text
http://localhost:3000
```

4. Para gerar a build de producao:

```bash
npm run build
```

## Estrutura

- `src/app/layout.jsx`: layout raiz do Next.
- `src/app/page.jsx`: tela principal React do gerador.
- `src/app/globals.css`: estilos globais sem imagem de background.
